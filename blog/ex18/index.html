Ex 18: 命名、变量、代码、函数

2014-Apr-03, 12:00:04, Thursday | Comments

这一节要接触函数了，用 def 关键字和 Python 真是像呢。话说，真的是松本看不惯 Python 才写的 Ruby 么？

定义了四个函数，第一个是用指针方式指定了参数个数，剩余三个则是指定了有限个参数个数。最后调用执行这四个函数。

  1. 函数定义是以 def 开始的；
  2. Ruby 的规范是以小写字符和 _ 下划线来进行连接；
  3. 函数名后紧接左括号；
  4. 多个参数时应用英文逗号隔开；
  5. 参数结束后要用英文右括号结束；
  6. 函数体要进行缩进两个空格(Delbert 自己更习惯用一个 Tab)；
  7. 与相应的 def 对齐的地方要有 end 来完成函数。
