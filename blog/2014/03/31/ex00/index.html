Ex 0: 准备工作

2014-Mar-31, 22:15:50, Monday | Comments

由于安装 Octopress，应该是已经部署好了 Ruby 的环境了。我是在 Windows 7 X64 环境下操作的。

Windows 7 X64

E:\SourceCode\GitHub> ruby -v
ruby 1.9.3p429 (2013-05-15) [i386-mingw32]

我安装的是 1.9.3 版本的。

同样，Windows 下也可以运行 Ruby 命令行 irb，可能 PowerShell 属于 bash 吧，退出方式是 Linux 的 Ctrl+D，而不是 Ctrl+Z（Ctrl+C 也不行）。

E:\SourceCode\GitHub> irb
irb(main):001:0> ^Z
=> nil
irb(main):002:0* ^C
irb(main):002:0>
E:\SourceCode\GitHub>

已经安装了 Vim 的，Notepad++ 也已经安装了 Vim 插件，不过还是用不惯。

暑假就可以用 Mac OS 了，期待我的更新吧。
